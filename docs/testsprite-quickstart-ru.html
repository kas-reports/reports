# Быстрый старт с TestSprite

TestSprite — это инструмент для автоматического тестирования, который позволяет разработчикам создавать, запускать и поддерживать тесты без написания тестового кода вручную.

---

## Установка

Добавьте TestSprite в ваш проект через npm:

```
npm install testsprite --save-dev
```

Или через yarn:

```
yarn add testsprite --dev
```

---

## Инициализация проекта

Запустите команду инициализации в корневой директории проекта:

```
npx testsprite init
```

Команда создаст файл конфигурации `testsprite.config.js` в корне проекта со следующим содержимым:

```javascript
module.exports = {
  // Целевой URL вашего приложения
  baseUrl: 'http://localhost:3000',

  // Директория для хранения скриншотов
  screenshotDir: './testsprite-screenshots',

  // Таймаут для каждого теста (в миллисекундах)
  timeout: 30000,

  // Режим запуска браузера (true = без интерфейса)
  headless: true,
};
```

---

## Первый тест

Создайте файл `my-first.test.js`:

```javascript
const { test, expect } = require('testsprite');

test('Главная страница загружается корректно', async ({ page }) => {
  await page.goto('/');
  await expect(page).toHaveTitle(/Мой сайт/);
  await expect(page.locator('h1')).toBeVisible();
});
```

Запустите тест:

```
npx testsprite run my-first.test.js
```

---

## Что выводит TestSprite

После запуска вы увидите:

```
✓ Главная страница загружается корректно (1.2s)

Результаты: 1 пройден, 0 провалено, 0 пропущено
Скриншоты сохранены в: ./testsprite-screenshots/
```

---

## Интерактивный режим

Для пошаговой отладки используйте интерактивный режим:

```
npx testsprite run --interactive my-first.test.js
```

В этом режиме тест останавливается перед каждым шагом, позволяя проверить состояние страницы.

---

## Запись тестов

TestSprite умеет записывать тесты на основе ваших действий в браузере:

```
npx testsprite record
```

Откроется браузер — выполните нужные действия на сайте. TestSprite автоматически сгенерирует тестовый файл.

## Настройки локали

Для тестирования локализованных интерфейсов укажите язык и регион:

```javascript
module.exports = {
  baseUrl: 'http://localhost:3000',
  locale: 'ru-RU',
  timezone: 'Europe/Moscow',
  // Формат дат в российской локали: ДД.ММ.ГГГГ
  dateFormat: 'DD.MM.YYYY',
};
```

**Важно для российской локали:**
- Даты отображаются в формате ДД.ММ.ГГГГ (не MM/DD/YYYY)
- Числа используют запятую как десятичный разделитель: 1 234,56
- Валюта: ₽ (рубль), размещается после числа: 1 500 ₽
- Часовой пояс по умолчанию: Europe/Moscow (UTC+3)

## Частые проблемы

**Тест не находит элемент:**
```javascript
// Используйте data-testid вместо CSS-классов
await page.locator('[data-testid="submit-button"]').click();
// Не рекомендуется:
await page.locator('.btn.btn-primary.submit').click();
```

**Кириллица в тестах:**
TestSprite полностью поддерживает Unicode. Тексты на кириллице в `expect()` работают корректно:
```javascript
await expect(page.locator('h1')).toHaveText('Добро пожаловать');
```

**Таймаут:**
Увеличьте таймаут для медленных операций:
```javascript
await page.locator('[data-testid="heavy-chart"]').waitFor({ timeout: 10000 });
```
